2) INSTALLATION DU LOGICIEL LEGO MINDSTORMS NXT

Avant d'installer le logiciel LEGO MINDSTORMS NXT, désinstallez les éventuelles versions antérieures installées sur votre système.

Vous devez posséder des droits d'administrateur pour pouvoir installer le logiciel LEGO MINDSTORMS NXT. 

Microsoft Windows
Pour installer le logiciel, exécutez « Autorun.exe » sur le CD-ROM, puis suivez les instructions qui s'affichent à l'écran.
Au terme de l'installation, il peut s'avérer nécessaire de redémarrer votre système. Lorsque le système a redémarré, double-cliquez sur l'icône Logiciel MINDSTORMS NXT qui se trouve sur votre bureau. Le logiciel LEGO MINDSTORMS NXT démarre. 

Apple Macintosh
Pour installer le logiciel, double-cliquez sur « Install » sur le DVD puis suivez les instructions qui s'affichent à l'écran.
Une fois le logiciel installé, sélectionnez Applications->LEGO Mindstorms NXT et cliquez sur Mindstorms NXT. Le logiciel LEGO MINDSTORMS NXT démarre. 

Le premier démarrage du logiciel LEGO MINDSTORMS NXT peut prendre quelques minutes. Ce délai de démarrage de l'application sera réduit au démarrage suivant.



3) DESINSTALLATION DU LOGICIEL LEGO MINDSTORMS NXT

Microsoft Windows XP
Cliquez sur Démarrer dans la barre des tâches de Windows, accédez à Tous les programmes->LEGO MINDSTORMS NXT->Désinstaller LEGO MINDSTORMS NXT. 
Vous pouvez aussi cliquer sur Démarrer dans la barre des tâches de Windows, accéder à Paramètres->Panneau de configuration, Ajout/Suppression de programmes, et cliquer sur Supprimer pour chaque élément LEGO MINDSTORMS NXT. Suivez les instructions qui s'affichent à l'écran. 

Microsoft Windows Vista
Cliquez sur Démarrer dans la barre des tâches de Windows, accédez à Tous les programmes->LEGO MINDSTORMS NXT->Désinstaller LEGO MINDSTORMS NXT. 
Vous pouvez aussi cliquer sur Démarrer dans la barre des tâches de Windows, accéder à Panneau de configuration, Programmes et fonctionnalités, et cliquer sur chaque élément LEGO MINDSTORMS NXT. Suivez les instructions qui s'affichent à l'écran. 

Microsoft Windows 7
Cliquez sur Démarrer dans la barre des tâches de Windows, accédez à Tous les programmes->LEGO MINDSTORMS NXT->Désinstaller LEGO MINDSTORMS NXT. 
Vous pouvez aussi cliquer sur Démarrer dans la barre des tâches de Windows, accéder à Panneau de configuration, Programmes et fonctionnalités, et cliquer sur chaque élément LEGO MINDSTORMS NXT. Suivez les instructions qui s'affichent à l'écran. 

Apple Mac 
Tirez le dossier du logiciel LEGO MINDSTORMS NXT des applications vers la corbeille pour le désinstaller.
