2) УСТАНОВКА ПРОГРАММНОГО ОБЕСПЕЧЕНИЯ LEGO MINDSTORMS NXT

Перед установкой программного обеспечения LEGO MINDSTORMS NXT удалите все установленные предыдущие версии.

Для установки LEGO MINDSTORMS NXT необходимо обладать правами администратора. 

Microsoft Windows
Для установки программного обеспечения запустите файл "Autorun.exe", находящийся на компакт-диске, и следуйте появляющимся на экране инструкциям.
После установки вам может потребоваться перезагрузить вашу систему. После перезагрузки системы дважды щелкните кнопкой мыши по иконке MINDSTORMS NXT на вашем рабочем столе. При этом запустится программа LEGO MINDSTORMS NXT. 

Apple Macintosh
Для установки программного обеспечения запустите файл «Install», находящийся на DVD-диске, и следуйте инструкциям, появляющимся на экране.
После установки выберите Applications->LEGO Mindstorms NXT и щелкните кнопкой мыши по надписи Mindstorms NXT. При этом запустится программа LEGO MINDSTORMS NXT. 

Первый запуск программного обеспечения LEGO MINDSTORMS NXT может длиться несколько минут. При следующем запуске программы время запуска уменьшится.



3) УДАЛЕНИЕ ПРОГРАММНОГО ОБЕСПЕЧЕНИЯ LEGO MINDSTORMS NXT

Microsoft Windows XP
Нажмите на панели задач Windows кнопку «Start» («Пуск»), затем выполните «All Programs->LEGO MINDSTORMS NXT->Uninstall LEGO MINDSTORMS NXT» («Все программы->LEGO MINDSTORMS NXT->Удалить LEGO MINDSTORMS NXT») 
Или нажмите кнопку «Start» («Пуск») на панели задач Windows, затем перейдите к «Settings->Control Panel» («Настройки->Панель управления»), «Add/Remove Programs» («Добавить/Удалить программы»), затем нажмите «Remove» («Удалить») для каждого объекта LEGO MINDSTORMS NXT. Следуйте инструкциям, появляющимся на экране. 

Microsoft Windows Vista
Нажмите на панели задач Windows кнопку «Start» («Пуск»), затем выполните «All Programs->LEGO MINDSTORMS NXT->Uninstall LEGO MINDSTORMS NXT» («Все программы->LEGO MINDSTORMS NXT->Удалить LEGO MINDSTORMS NXT») 
Или нажмите кнопку «Start» («Пуск») на панели задач Windows, затем перейдите к «Control Panel» («Панель управления»), «Uninstall» («Удалить»), затем выберите все объекты LEGO MINDSTORMS NXT. Следуйте инструкциям, появляющимся на экране. 

Microsoft Windows 7
Нажмите на панели задач Windows кнопку «Start» («Пуск»), затем выполните «All Programs->LEGO MINDSTORMS NXT->Uninstall LEGO MINDSTORMS NXT» («Все программы->LEGO MINDSTORMS NXT->Удалить LEGO MINDSTORMS NXT») 
Или нажмите кнопку «Start» («Пуск») на панели задач Windows, затем перейдите к «Control Panel» («Панель управления»), «Uninstall» («Удалить»), затем выберите все объекты LEGO MINDSTORMS NXT. Следуйте инструкциям, появляющимся на экране. 

Apple Mac 
Для удаления перетащите папку с программным обеспечением LEGO MINDSTORMS NXT из Applications («Приложения») в корзину.
