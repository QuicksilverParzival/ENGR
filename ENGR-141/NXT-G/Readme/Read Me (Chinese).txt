2）安装 LEGO MINDSTORMS NXT 软件。

在安装LEGO MINDSTORMS NXT 软件之前，请卸载以前安装的版本。

安装 LEGO MINDSTORMS NXT 软件时需要管理员授权。

微软 Windows
安装软件时，在CD-ROM上运行“自动运行（Autorun.exe）”并按照屏幕上的指示操作。
安装时，您可能需要重启系统。当系统重启时，双击电脑屏幕上的 MINDSTORMS NXT 软件图标。这将启动 LEGO MINDSTORMS NXT 软件。

苹果 Macintosh
安装软件时，在 DVD 上双击“安装（Install）”并按照屏幕上的指示操作。
在安装时，选择 Applications->LEGO Mindstorms NXT，然后点击 Mindstorms NXT。这将启动 LEGO MINDSTORMS NXT 软件。

第一次启动 LEGO MINDSTORMS NXT 软件时，可能会花费几分钟。启动时间在您下次打开应用程序时会减少。



3）卸载 LEGO MINDSTORMS NXT 软件。

微软 Windows XP
在 Windows 任务栏中点击开始（start），然后按照所有程序（All Programs）->LEGO MINDSTORMS NXT->卸载 LEGO MINDSTORMS NXT 依次进行。
您还可以在 Windows 任务栏中点击开始（start），然后按照设置（Settings）->控制面板（Control Panel），添加/删除所有程序（Add/Remove Programs），然后点击删除（Remove）LEGO MINDSTORMS NXT 的每个项目。按照屏幕上的指示操作。

微软 Windows Vista
在 Windows 任务栏中点击开始（start），然后按照所有程序（All Programs）->LEGO MINDSTORMS NXT->卸载 LEGO MINDSTORMS NXT 依次进行。
您还可以在 Windows 任务栏中点击开始（start），然后在控制面板（Control Panel）上卸载一个程序，点击 LEGO MINDSTORMS NXT 的每个项目。按照屏幕上的指示操作。

微软 Windows 7
在 Windows 任务栏中点击开始（start），然后按照所有程序（All Programs）->LEGO MINDSTORMS NXT->卸载 LEGO MINDSTORMS NXT 依次进行。
您还可以在 Windows 任务栏中点击开始（start），然后在控制面板（Control Panel）上卸载一个程序，点击 LEGO MINDSTORMS NXT 的每个项目。按照屏幕上的指示操作。

苹果 Mac 电脑 
将 LEGO MINDSTORMS NXT 软件文件夹从应用程序类拖到到垃圾箱内。
