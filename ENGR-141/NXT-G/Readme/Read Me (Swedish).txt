2) INSTALLERA PROGRAMVARAN LEGO MINDSTORMS NXT

Innan du installerar programvaran LEGO MINDSTORMS NXT måste du avinstallera alla tidigare installerade versioner.

Administratörsrättigheter krävs för installation av programvaran LEGO MINDSTORMS NXT. 

Microsoft Windows
För att installera programvaran kör du ”Autorun.exe” på CD-ROM-skivan och följer sedan instruktionerna på skärmen.
Efter installationen kan du behöva starta om systemet. När du startat om systemet dubbelklickar du på ikonen för programvaran MINDSTORMS NXT på skärmen. Då startar programvaran LEGO MINDSTORMS NXT. 

Apple Macintosh
För att installera programvaran dubbelklickar du på ”Installera” på DVD-skivan och följer sedan instruktionerna på skärmen.
När programvaran är installerad väljer du Program -> LEGO Mindstorms NXT och klickar på Mindstorms NXT. Då startar programvaran LEGO MINDSTORMS NXT. 

Första gången du startar programvaran LEGO MINDSTORMS NXT kan det ta några minuter. Uppstartstiden minskar nästa gång du startar programmet.



3) AVINSTALLERA PROGRAMVARAN LEGO MINDSTORMS NXT

Microsoft Windows XP
Klicka på Start på Windows aktivitetsfält, gå till Alla program -> LEGO MINDSTORMS NXT -> Avinstallera LEGO MINDSTORMS NXT. 
Du kan också klicka på Start i Windows aktivitetsfält, gå till Inställningar -> Kontrollpanelen, Lägg till eller ta bort program och klicka på ”Ta bort” för varje programvara som hör till LEGO MINDSTORMS NXT. Följ instruktionerna på skärmen. 

Microsoft Windows Vista
Klicka på Start på Windows aktivitetsfält, gå till Alla program -> LEGO MINDSTORMS NXT -> Avinstallera LEGO MINDSTORMS NXT. 
Du kan också klicka på Start i Windows aktivitetsfält, gå till Kontrollpanelen, Avinstallera ett program och klicka på varje programvara som hör till LEGO MINDSTORMS NXT. Följ instruktionerna på skärmen. 

Microsoft Windows 7
Klicka på Start på Windows aktivitetsfält, gå till Alla program -> LEGO MINDSTORMS NXT -> Avinstallera LEGO MINDSTORMS NXT. 
Du kan också klicka på Start i Windows aktivitetsfält, gå till Kontrollpanelen, Avinstallera program och klicka på varje programvara som hör till LEGO MINDSTORMS NXT. Följ instruktionerna på skärmen. 

Apple Mac 
Dra programmappen med LEGO MINDSTORMS NXT från Program till papperskorgen för att avinstallera.
