2) INSTALACIÓN DEL SOFTWARE LEGO MINDSTORMS NXT

Antes de instalar el software LEGO MINDSTORMS NXT, desinstale cualquier versión anterior instalada.

Para instalar el software LEGO MINDSTORMS NXT necesitará disponer de privilegios de administrador. 

Microsoft Windows
Para instalar el software, ejecute el archivo "Autorun.exe" que encontrará en el CD-ROM y siga las instrucciones que aparecerán en pantalla.
Es posible que deba reiniciar el sistema una vez finalizada la instalación. Tras reiniciarlo, haga doble clic en el icono del software MINDSTORMS NXT situado en el escritorio. Al hacerlo, se iniciará el software LEGO MINDSTORMS NXT. 

Apple Macintosh
Para instalar el software, haga doble clic en el archivo "Install" que encontrará en el DVD y siga las instrucciones que aparecerán en la pantalla.
Una vez instalado el software, seleccione Aplicaciones -> LEGO Mindstorms NXT y haga clic en Mindstorms NXT. Al hacerlo, se iniciará el software LEGO MINDSTORMS NXT. 

Es posible que el primer inicio del software LEGO MINDSTORMS NXT se prolongue durante algunos minutos. El tiempo de arranque será inferior la próxima vez que inicie la aplicación.



3) DESINSTALACIÓN DEL SOFTWARE LEGO MINDSTORMS NXT

Microsoft Windows XP
Haga clic en el menú Inicio a través de la barra de tareas de Windows y seleccione Todos los programas -> LEGO MINDSTORMS NXT -> Desinstalar LEGO MINDSTORMS NXT. 
O bien, haga clic en el menú Inicio a través de la barra de tareas de Windows y seleccione Configuración -> Panel de control, Agregar o quitar programas. A continuación, haga clic en Quitar por cada uno de los elementos pertenecientes a LEGO MINDSTORMS NXT. Siga las instrucciones que aparecerán en la pantalla. 

Microsoft Windows Vista
Haga clic en el menú Inicio a través de la barra de tareas de Windows y seleccione Todos los programas -> LEGO MINDSTORMS NXT -> Desinstalar LEGO MINDSTORMS NXT. 
O bien, haga clic en el menú Inicio a través de la barra de tareas de Windows y acceda a Panel de control, Desinstalar un programa. A continuación, haga clic en cada uno de los elementos pertenecientes a LEGO MINDSTORMS NXT. Siga las instrucciones que aparecerán en la pantalla. 

Microsoft Windows 7
Haga clic en el menú Inicio a través de la barra de tareas de Windows y seleccione Todos los programas -> LEGO MINDSTORMS NXT -> Desinstalar LEGO MINDSTORMS NXT. 
O bien, haga clic en el menú Inicio a través de la barra de tareas de Windows y acceda a Panel de control, Desinstalar un programa. A continuación, haga clic en cada uno de los elementos pertenecientes a LEGO MINDSTORMS NXT. Siga las instrucciones que aparecerán en la pantalla. 

Apple Mac 
Arrastre la carpeta del software LEGO MINDSTORMS NXT desde la carpeta Aplicaciones hasta la Papelera para llevar a cabo la desinstalación.
