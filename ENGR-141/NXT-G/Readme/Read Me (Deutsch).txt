2) INSTALLATION DER LEGO MINDSTORMS NXT SOFTWARE

Vor der Installation der LEGO MINDSTORMS NXT Software sollten Sie installierte frühere Versionen deinstallieren.

Für die Installation der LEGO MINDSTORMS NXT Software sind Administratorrechte erforderlich. 

Microsoft Windows
Starten Sie die Datei „Autorun.exe“ auf der CD-ROM, um mit der Installation zu beginnen, und folgen Sie den Bildschirmanweisungen.
Nach Abschluss der Installation müssen Sie Ihr System u. U. neu starten. Doppelklicken Sie im Anschluss daran das MINDSTORMS NXT Software Icon auf Ihrem Desktop, um die LEGO MINDSTORMS NXT Software zu starten. 

Apple Macintosh
Auf der DVD befindet sich die Datei „Install“. Führen Sie auf diese einen Doppelklick aus, um mit der Installation zu beginnen, und folgen Sie den Bildschirmanweisungen.
Wählen Sie nach der Installation „Applications“ ->LEGO Mindstorms NXT und klicken dann auf Mindstorms NXT, um die LEGO MINDSTORMS NXT Software zu starten. 

Der erste Start der LEGO MINDSTORMS NXT Software kann ein paar Minuten in Anspruch nehmen; diese Wartezeit wird sich beim nächsten Start der Applikation verringern.



3) DEINSTALLATION DER LEGO MINDSTORMS NXT SOFTWARE

Microsoft Windows XP
Klicken Sie in der Windows-Taskleiste auf Start, gehen Sie auf Programme -> LEGO MINDSTORMS NXT -> Uninstall LEGO MINDSTORMS NXT (LEGO MINDSTORMS NXT deinstallieren). 
Alternativ können Sie auch in der Windows-Taskleiste auf Start klicken, auf Einstellungen -> Systemsteuerung -> Software gehen, alle angezeigten LEGO MINDSTORMS NXT Elemente auswählen, auf Entfernen klicken und den Bildschirmanweisungen folgen. 

Microsoft Windows Vista
Klicken Sie in der Windows-Taskleiste auf Start, gehen Sie auf Programme -> LEGO MINDSTORMS NXT -> Uninstall LEGO MINDSTORMS NXT (LEGO MINDSTORMS NXT deinstallieren). 
Alternativ können Sie auch in der Windows-Taskleiste auf Start klicken, auf Einstellungen -> Systemsteuerung -> Software gehen, alle angezeigten LEGO MINDSTORMS NXT Elemente auswählen, auf Entfernen klicken und den Bildschirmanweisungen folgen. 

Microsoft Windows 7
Klicken Sie in der Windows-Taskleiste auf Start, gehen Sie auf Programme -> LEGO MINDSTORMS NXT -> Uninstall LEGO MINDSTORMS NXT (LEGO MINDSTORMS NXT deinstallieren). 
Alternativ können Sie auch in der Windows-Taskleiste auf Start klicken, auf Einstellungen -> Systemsteuerung -> Software gehen, alle angezeigten LEGO MINDSTORMS NXT Elemente auswählen, auf Entfernen klicken und den Bildschirmanweisungen folgen. 

Apple Mac 
Ziehen Sie den LEGO MINDSTORMS NXT Software-Ordner in den Papierkorb. Damit erfolgt die Deinstallation.
