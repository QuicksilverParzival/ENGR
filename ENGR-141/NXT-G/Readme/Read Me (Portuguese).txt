2) INSTALANDO O PROGRAMA LEGO MINDSTORMS NXT

Antes de instalar o programa LEGO MINDSTORMS NXT, desinstale qualquer versão  instalada anteriormente.

É necessário possuir privilégios de administrador ao instalar o programa LEGO MINDSTORMS NXT. 

Microsoft Windows
Para instalar o programa execute o "Autorun.exe" no CD-ROM e siga as instruções na tela.
Ao concluir a instalação será necessário reiniciar o sistema. Depois que o sistema for reiniciado, dê dois cliques no ícone do programa MINDSTORMS NXT na área de trabalho. Isso iniciará o programa LEGO MINDSTORMS NXT. 

Apple Macintosh
Para instalar o programa clique duas vezes em "Instalar" no DVD e siga as instruções na tela.
Após a instalação, selecione Aplicações->LEGO Mindstorms NXT e clique em Mindstorms NXT. Isso iniciará o programa LEGO MINDSTORMS NXT. 

Alguns minutos de espera poderão ser necessários para iniciar o LEGO MINDSTORMS NXT pela primeira vez. O tempo de partida diminuirá da próxima vez que a aplicação for iniciada.



3) DESINSTALANDO O PROGRAMA LEGO MINDSTORMS NXT

Microsoft Windows XP
Clique em Iniciar, na barra de tarefas do Windows, vá para Todos os Programas->LEGO MINDSTORMS NXT->Desinstalar LEGO MINDSTORMS NXT. 
Uma alternativa é clicar em Iniciar, na barra de tarefas do Windows, e ir para Configurações->Painel de Controle, Adicionar/Remover Programas e clicar em Remover para cada item LEGO MINDSTORMS NXT. Sigas as instruções na tela. 

Microsoft Windows Vista
Clique em Iniciar, na barra de tarefas do Windows, vá para Todos os Programas->LEGO MINDSTORMS NXT->Desinstalar LEGO MINDSTORMS NXT. 
Uma alternativa é clicar em Iniciar, na barra de tarefas do Windows, e ir para Painel de Controle, Desinstalar Programa e clicar em cada item LEGO MINDSTORMS NXT. Sigas as instruções na tela. 

Microsoft Windows 7
Clique em Iniciar, na barra de tarefas do Windows, vá para Todos os Programas->LEGO MINDSTORMS NXT->Desinstalar LEGO MINDSTORMS NXT. 
Uma alternativa é clicar em Iniciar, na barra de tarefas do Windows, e ir para Painel de Controle, Desinstalar Programa e clicar em cada item LEGO MINDSTORMS NXT. Sigas as instruções na tela. 

Apple Mac 
Arraste a pasta do programa LEGO MINDSTORMS NXT de Aplicações para a Lixeira para desinstalar.
