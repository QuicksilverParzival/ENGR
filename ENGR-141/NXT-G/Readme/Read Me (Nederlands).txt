2) INSTALLEREN VAN DE LEGO MINDSTORMS NXT SOFTWARE

Verwijder alle eerdere versies van de software alvorens deze versie van de LEGO MINDSTORMS NXT Software te installeren.

Om de LEGO MINDSTORMS NXT software te kunnen installeren, zijn Administrator bevoegdheden vereist. 

Microsoft Windows
Om de software te installeren: activeer "Autorun.exe" op de CD-ROM en volg de instructies op het scherm.
Na de installatie moet de computer misschien opnieuw gestart worden. Nadat de computer opnieuw gestart is: dubbelklik op de MINDSTORMS NXT Software-icoon op het bureaublad. De LEGO MINDSTORMS NXT Software zal nu gestart worden. 

Apple Macintosh
Om de software te installeren: dubbelklik op "Install" op de dvd en volg daarna de instructies op het scherm.
Ga na installatie naar "Toepassingen (Applications)->LEGO Mindstorms NXT" en klik op Mindstorms NXT. De LEGO MINDSTORMS NXT software zal nu gestart worden. 

Als de software voor het eerst gebruikt wordt, kan de procedure een paar minuten duren. Na de eerste keer zal de startprocedure sneller verlopen.



3) VERWIJDEREN VAN DE LEGO MINDSTORMS NXT SOFTWARE

Microsoft Windows XP
Klik op "Start" in de Windows taakbalk en ga naar "Alle Programma’s->LEGO MINDSTORMS NXT->Uninstall" (Verwijder) LEGO MINDSTORMS NXT. 
Als alternatief is het mogelijk op "Start" te klikken in de Windows taakbalk en naar "Settings" (Instellingen) en "Control Panel" (Configuratiescherm) te gaan. Kies dan "Add/Remove Programs" (Programma’s toevoegen/verwijderen), selecteer de LEGO MINDSTORMS NXT items en klik op "Remove" (Verwijderen). Volg de instructies op het scherm. 

Microsoft Windows Vista
Klik op "Start" in de Windows taakbalk en ga naar "Alle Programma’s->LEGO MINDSTORMS NXT->Uninstall" (Verwijder) LEGO MINDSTORMS NXT. 
Als alternatief is het mogelijk op "Start" te klikken in de Windows taakbalk en naar "Control Panel" (Configuratiescherm) te gaan. Kies dan "Uninstall a Program" (Programma verwijderen) en klik op de LEGO MINDSTORMS NXT items. Volg de instructies op het scherm. 

Microsoft Windows 7
Klik op "Start" in de Windows taakbalk en ga naar "Alle Programma’s->LEGO MINDSTORMS NXT->Uninstall" (Verwijder) LEGO MINDSTORMS NXT. 
Als alternatief is het mogelijk op "Start" te klikken in de Windows taakbalk en naar "Control Panel" (Configuratiescherm) te gaan. Kies dan "Uninstall a Program" (Programma verwijderen) en klik op de LEGO MINDSTORMS NXT items. Volg de instructies op het scherm. 

Apple Mac 
Trek de map met de LEGO MINDSTORMS NXT software vanuit de "Programma’s" (Applications) map naar de Vuilnisbak (Trash Can) om het programma te verwijderen.
