2) INSTALLATION AF LEGO MINDSTORMS NXT SOFTWARE

Afinstaller alle tidligere versioner, inden LEGO MINDSTORMS NXT Software installeres.

Installation af LEGO MINDSTORMS NXT Software kræver administratorrettigheder.

Microsoft Windows
Kør "Autorun.exe" på cd’en for at installere softwaren, og følg vejledningen på skærmen.
Når softwaren er installeret, skal du muligvis genstarte computeren. Dobbeltklik på skrivebordsikonet for MINDSTORMS NXT Software, når systemet er blevet genstartet. Dette starter LEGO MINDSTORMS NXT Softwaren.

Apple Macintosh
Dobbeltklik på "Install" på dvd'en for at installere softwaren, og følg vejledningen på skærmen.
Vælg Applikationer -> LEGO Mindstorms NXT, og klik på Mindstorms NXT, når softwaren er installeret. Dette starter LEGO MINDSTORMS NXT Softwaren.

Første gang du starter LEGO MINDSTORMS NXT Software kan det tage et par minutter. Opstartstiden vil være kortere, næste gang programmet startes.



3) AFINSTALLATION AF LEGO MINDSTORMS NXT SOFTWARE

Microsoft Windows XP
Klik på Start på Windows-proceslinjen, og vælg Alle Programmer -> LEGO MINDSTORMS NXT -> Afinstaller LEGO MINDSTORMS NXT.
Eller klik på Start på Windows-proceslinjen, vælg Indstillinger -> Kontrolpanel, Tilføj/fjern programmer, og klik på Fjern ud for hvert LEGO MINDSTORMS NXT produkt. Følg vejledningen på skærmen.

Microsoft Windows Vista
Klik på Start på Windows-proceslinjen, og vælg Alle Programmer -> LEGO MINDSTORMS NXT -> Afinstaller LEGO MINDSTORMS NXT.
Eller klik på Start på Windows-proceslinjen, vælg Kontrolpanel, Programmer og funktioner, og klik på Fjern ud for hvert LEGO MINDSTORMS NXT produkt. Følg vejledningen på skærmen.

Microsoft Windows 7
Klik på Start på Windows-proceslinjen, og vælg Alle Programmer -> LEGO MINDSTORMS NXT -> Afinstaller LEGO MINDSTORMS NXT.
Eller klik på Start på Windows-proceslinjen, vælg Kontrolpanel, Programmer og funktioner, og klik på Fjern ud for hvert LEGO MINDSTORMS NXT produkt. Følg vejledningen på skærmen.

Apple Mac
Træk mappen med LEGO MINDSTORMS NXT softwaren fra Programmer til Papirkurven for at afinstallere.
