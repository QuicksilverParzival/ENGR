2) INSTALLAZIONE DEL SOFTWARE LEGO MINDSTORMS NXT

Prima di installare il Software LEGO MINDSTORMS NXT, disinstallare qualsiasi versione precedente.

Quando si installa il Software LEGO MINDSTORMS NXT sono richiesti i privilegi di Amministratore. 

Microsoft Windows
Per installare il Software eseguire “Autorun.exe” sul CD-ROM e seguire le istruzioni sullo schermo.
Una volta completata l’installazione sarà necessario riavviare il sistema. Dopo il riavvio, fare doppio clic sull’icona del Software MINDSTORMS NXT sul desktop. In questo modo si avvia il Software LEGO MINDSTORMS NXT. 

Apple Macintosh
Per installare il Software fare doppio clic su “Install” sul DVD e seguire le istruzioni sullo schermo.
Una volta completata l’installazione, selezionare Applicazioni->LEGO Mindstorms NXT e fare clic su Mindstorms NXT. In questo modo si avvia il Software LEGO MINDSTORMS NXT. 

La prima volta in cui si avvia il Software LEGO MINDSTORMS NXT potrebbero essere necessari alcuni minuti. Il tempo di avvio diminuisce al prossimo avvio dell’applicazione.



3) DISINSTALLAZIONE DEL SOFTWARE LEGO MINDSTORMS NXT

Microsoft Windows XP
Fare clic su Start dalla barra di Windows, aprire Tutti i programmi->LEGO MINDSTORMS NXT->Disinstalla LEGO MINDSTORMS NXT. 
In alternativa, fare clic su Start dalla barra di Windows, aprire Impostazioni->Pannello di controllo, Installazione applicazioni e fare clic su Rimuovi per ogni voce di LEGO MINDSTORMS NXT. Seguire le istruzioni sullo schermo. 

Microsoft Windows Vista
Fare clic su Start dalla barra di Windows, aprire Tutti i programmi->LEGO MINDSTORMS NXT->Disinstalla LEGO MINDSTORMS NXT. 
In alternativa, fare clic su Start dalla barra di Windows, aprire il Pannello di controllo->Disinstalla un programma e fare clic su ogni voce di LEGO MINDSTORMS NXT. Seguire le istruzioni sullo schermo. 

Microsoft Windows 7
Fare clic su Start dalla barra di Windows, aprire Tutti i programmi->LEGO MINDSTORMS NXT->Disinstalla LEGO MINDSTORMS NXT. 
In alternativa, fare clic su Start dalla barra di Windows, aprire il Pannello di controllo->Disinstalla un programma e fare clic su ogni voce di LEGO MINDSTORMS NXT. Seguire le istruzioni sullo schermo. 

Apple Mac 
Trascinare la cartella del Software LEGO MINDSTORMS NXT dalle Applicazioni al Cestino per disinstallarlo.
