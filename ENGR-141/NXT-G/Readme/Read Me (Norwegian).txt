2) INSTALLASJON AV LEGO MINDSTORMS NXT PROGRAMVARE

Før du installerer LEGO MINDSTORMS NXT programvare, må du avinstallere eventuell tidligere installert versjon.

Det kreves administratorrettigheter for å installere LEGO MINDSTORMS NXT programvare. 

Microsoft Windows
Du installerer programvaren ved å kjøre "Autorun.exe" på CD-ROM-en og følge instruksjonene på skjermen.
Etter installasjonen må du muligens starte systemet på nytt. Når systemet har startet på nytt, dobbeltklikker du på ikonet for MINDSTORMS NXT programvare på skrivebordet. Dette vil starte LEGO MINDSTORMS NXT programvare. 

Apple Macintosh
Installer programvaren ved å dobbeltklikke på "Install" ("installer") på DVD-en og følg instruksjonene på skjermen.
Når programvaren er installert, velger du Applications->LEGO Mindstorms NXT og klikker Mindstorms NXT. Dette vil starte LEGO MINDSTORMS NXT programvare. 

Første gang du starter LEGO MINDSTORMS NXT programvare kan det ta noen minutter. Oppstartstiden vil være kortere neste gang du starter applikasjonen.



3) AVINSTALLASJON AV LEGO MINDSTORMS NXT PROGRAMVARE

Microsoft Windows XP
Klikk Start på oppgavelinjen i Windows, gå til Alle programmer->LEGO MINDSTORMS NXT->Avinstaller LEGO MINDSTORMS NXT. 
Alternativt kan du klikke Start fra oppgavelinjen i Windows, gå til Kontrollpanel>Legg til/fjern programmer, og klikke Fjern for hvert enkelt LEGO MINDSTORMS NXT emne. Følg instruksjonene på skjermen. 

Microsoft Windows Vista
Klikk Start på oppgavelinjen i Windows, gå til Alle programmer->LEGO MINDSTORMS NXT->Avinstaller LEGO MINDSTORMS NXT. 
Alternativt kan du klikke Start fra oppgavelinjen i Windows, gå til Kontrollpanel>Avinstaller programmer, og klikke på hvert enkelt LEGO MINDSTORMS NXT emne. Følg instruksjonene på skjermen. 

Microsoft Windows 7
Klikk Start på oppgavelinjen i Windows, gå til Alle programmer->LEGO MINDSTORMS NXT->Avinstaller LEGO MINDSTORMS NXT. 
Alternativt kan du klikke Start fra oppgavelinjen i Windows, gå til Kontrollpanel>Avinstaller programmer, og klikke på hvert enkelt LEGO MINDSTORMS NXT emne. Følg instruksjonene på skjermen. 

Apple Mac 
Dra programvaremappen for LEGO MINDSTORMS NXT fra Programmer til Papirkurven for å avinstallere.
