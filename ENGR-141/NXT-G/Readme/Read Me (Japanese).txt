2) レゴ マインドストーム NXT ソフトウェアのインストール

レゴ マインドストーム NXT ソフトウェアをインストールする前に、旧バージョンのものがインストールされている場合は、それをアンインストールしてください。

レゴ マインドストーム NXT ソフトウェアをインストールするためには、管理者権限が必要となります。 

Microsoft Windows
ソフトウェアをインストールするためには、 CD-ROM に搭載されている「Autorun.exe」を実行して、画面に表示される指示に従います。
インストールが完了すると、システムを再起動しなければならない場合もあります。システムが再起動したら、デスクトップ上にあるマインドストーム NXT ソフトウェアのアイコンをダブルクリックします。これにより、レゴ マインドストーム NXT ソフトウェアが起動します。 

アップル Macintosh
ソフトウェアをインストールするためには、 DVD に搭載されている「Install」をダブルクリックして、画面に表示される指示に従います。
インストールが完了したら、[アプリケーション] -> [レゴ マインドストーム NXT] で [マインドストーム NXT] をクリックします。これにより、レゴ マインドストーム NXT ソフトウェアが起動します。 

レゴ マインドストーム NXT ソフトウェアを始めて起動する際には、2 分程度時間がかかる場合があります。次回アプリケーションを起動する際には少ない時間で立ち上がります。



3) レゴ マインドストーム NXT ソフトウェアのアンインストール

Microsoft Windows XP
Windows タスク バーの [Start] をクリックして、[すべてのプログラム] -> [レゴ マインドストーム NXT] -> [レゴ マインドストーム NXT のアンインストール]をクリックします。 
または、Windows タスク バーの [Start] をクリックして、[コントロール パネル] -> [プログラムの追加と削除] で、レゴ マインドストーム NXT の項目全てにチェックを入れて削除します。画面に表される指示に従います。 

Microsoft Windows Vista
Windows タスク バーの [Start] をクリックして、[すべてのプログラム] -> [レゴ マインドストーム NXT] -> [レゴ マインドストーム NXT のアンインストール]をクリックします。 
または、Windows タスク バーの [Start] をクリックして、[コントロール パネル] -> [プログラムの追加と削除]で、レゴ マインドストーム NXT の項目全てにチェックを入れて削除します。画面に表u12373 される指示に従います。 

Microsoft Windows 7
Windows タスク バーの [Start] をクリックして、[すべてのプログラム] -> [レゴ マインドストーム NXT] -> [レゴ マインドストーム NXT のアンインストール]をクリックします。 
または、Windows タスク バーの [Start] をクリックして、[コントロール パネル] -> [プログラムの追加と削除]で、レゴ マインドストーム NXT の項目全てにチェックを入れて削除します。画面に表れる指示に従います。 

アップル Mac 
アンインストールするには、レゴ マインドストーム NXT ソフトウェアのフォルダを [アプリケーション] からゴミ箱にドラッグして入れます。
